# Docker 명령어

위 명령어는 윈도우 기준입니다.

#### 1. 이미지 확인

```
docker images
```

#### 2. 이미지 삭제

```
docker rmi <이미지 ID>
```

```
docker rmi -f <이미지 ID>
```

#### 3. 컨테이너 실행중 stop

```
docker <컨테이너 이름 혹은 아이디>
```

#### 4. 컨테이너 상태 확인

실행 중인 컨테이너의 상태를 확인할 때에는 ps 명령어를 사용합니다.

호스트에 존재하는 모든 컨테이너를 확인할 때에는 ps 명령어와 함께 -a 옵션을 사용합니다.

```
docker ps
```

```
docker ps -a
```
