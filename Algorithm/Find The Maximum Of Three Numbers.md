# 세 개의 숫자 중 최댓값 찾기

## 문제

3 개의 정수가 주어질 때, 이 중 최댓값을 찾는 프로그램을 작성하여라.

## 입력

첫째 줄에 3개의 정수a,b,c 가 주어진다. (0≤a,b,c≤10,000)

## 출력

첫째 줄에 a,b,c 중 최댓값을 출력한다.

-----

### 예제 입력

10 2 5

### 예제 출력

10

--------

### 예제 입력

3 7 7

### 예제 출력

7

```c++
#include <stdio.h>

int main() {
  int a[3] = {0};
  for(int i = 0; i < 3; i++){
    scanf("%d", &a[i]);
  }

  int max = 0;
  for(int i = 0; i < 3; i++){
    if(max < a[i]){
      max = a[i];
    }
  }
  
  printf("%d\n", max);
  
  return 0;
}
```
