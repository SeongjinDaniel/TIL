# 테이블 컬럼 추가 수정 삭제 



## 추가

### 컬럼 추가

```sql
ALTER TABLE [테이블명] ADD [컬럼명] [타입] [옵션]; 
```





## 수정

### 컬럼명 변경 및 타입 수정

```sql
ALTER TABLE [테이블명] CHANGE [컬럼명] [변경할컬럼명] [수정할타입];
```



### 컬럼 타입 수정

```sql
ALTER TABLE [테이블명] MODIFY [컬럼명] [수정할타입];
```



### 컬럼 디폴트값 수정

```sql
ALTER TABLE [테이블명] ALTER COLUMN [변경할컬럼명] SET DEFAULT [디폴트값];
```





## 삭제

### 컬럼 삭제

```sql
ALTER TABLE [테이블명] DROP [컬럼명];
```
