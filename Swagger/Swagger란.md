# Swagger란?



REST API 문서화를 도와주는 툴이다.

문서를 아무리 잘 만들었다고 해도 파라미터나 주소가 변경될 경우 문서와 싱크가 안맞을 수도 있다. 그런 문제를 해결해 주는 것이 Swagger이다.
