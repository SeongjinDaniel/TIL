

## 리눅스 명령어 / tr 명령어 - 문자 변환/삭제

 

tr 명령어는 지정한 문자를 변환하거나 삭제하는 명령어이다. 특정한 문자를 다른 문자로 변환하거나 특정 문자를 제거하는데 사용되는 명령어이다. 

 

### 명령어 사용법

`tr [옵션] 문자열1 [문자열2]`

| -d   | 문자열 1에서 지정한 문자를 삭제합니다.   |
| ---- | ---------------------------------------- |
| -s   | 문자열 2에서 반복되는 문자를 삭제합니다. |
| -t   | 문자열1을 문자열2의 길이로 자릅니다.     |
